# IntelliStudio - AI 智能创作平台

## 🚀 快速开始

### 方式一：一键启动（推荐）
直接双击 `启动.bat` 文件，即可自动启动应用！

### 方式二：手动启动
```bash
# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

然后在浏览器中打开：http://localhost:3000

## ✨ 功能特点

### 🤖 智能自动化
- **一句话创作**：只需用自然语言描述你的需求
- **自动工作流**：AI 自动完成从剧本到成品的全流程
- **智能分析**：自动理解需求，拆解任务

### 🎬 短剧创作流程
1. **需求分析** - AI 理解你的创作意图
2. **剧本生成** - 自动创作完整剧本
3. **角色设计** - 设计角色形象和性格
4. **视频生成** - 自动生成视频画面
5. **声音克隆** - 生成角色配音
6. **字幕生成** - 自动匹配字幕
7. **导出成品** - 一键下载最终视频

### 🎨 界面特色
- **三种主题**：极光蓝、暗夜紫、纯黑 AMOLED
- **流畅动画**：Framer Motion 丝滑过渡
- **玻璃态设计**：现代磨砂玻璃效果
- **动态背景**：Canvas 粒子动画

## 📖 使用指南

### 创建新项目
1. 在左侧选择快速模板，或点击"新建项目"
2. 在主界面输入框中描述你的需求
3. 点击"开始创作"或按 Enter 键
4. 等待 AI 自动完成所有工作
5. 完成后点击"下载成品"

```
我想要一个关于都市爱情的短剧，女主角是咖啡店老板，男主角是建筑师，两人在雨中相遇...

```
悬疑推理短剧，侦探调查一起密室杀人案...

## 🛠️ 技术栈

- **React 18** - 前端框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **Framer Motion** - 动画库
- **Zustand** - 状态管理
- **i18next** - 国际化支持

## 🎯 核心优势

1. **零门槛**：不需要专业技能，会说话就能创作
2. **全自动化**：AI 完成所有技术工作
3. **本地优先**：所有数据保存在本地
4. **多语言**：支持中文、维吾尔语等
5. **可扩展**：工作流系统支持自定义节点

## 🔧 常见问题

### 端口被占用？
修改 `vite.config.ts` 中的端口配置

### 如何切换主题？
点击右上角设置图标，选择你喜欢的主题

### 支持哪些语言？
目前支持中文（简体）和维吾尔语，后续会添加更多语言
